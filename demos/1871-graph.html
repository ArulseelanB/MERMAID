Theme Variables

The marker stroke should be from the theme variable: 
      %%{init:{"theme":"base", "themeVariables": {"lineColor":"red"}}}%%
      graph LR
      A <--> B x--x C o--o D
    
      %%{init:{"theme":"base", "themeVariables": {"lineColor":"blue"}}}%%
      graph LR
      A <--> B x--x C o--o D
