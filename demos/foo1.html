Theme Variables

The marker stroke should be from the theme variable: 
    %%{init:{"theme":"base", "themeVariables": {"lineColor":"red"}}}%%
    flowchart LR
    A --> B
%%{init:{"theme":"base", "themeVariables": {"lineColor":"red"}}}%% flowchart LR A --> B 
    %%{init:{"theme":"base", "themeVariables": {"lineColor":"blue"}}}%%
    flowchart LR
    A -->|marker should be blue| B
%%{init:{"theme":"base", "themeVariables": {"lineColor":"blue"}}}%% flowchart LR A -->|marker should be blue| B
